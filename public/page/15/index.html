Linux Cheatsheet

2011/11/29

Navigation and File Management: ðŸ”—

  1. ls - List directory contents (-l for long format, -a for all files including hidden)
  2. cd - Change directory (.. for parent directory, ~ for home directory)
  3. pwd - Print working directory
  4. mkdir - Create a directory (-p to create parent directories)
  5. rmdir - Remove an empty directory
  6. rm - Remove files or directories (-r for recursive, -f for force)
  7. cp - Copy files or directories (-r for recursive, -i for interactive)
  8. mv - Move or rename files or directories
  9. touch - Create an empty file or update file timestamps
  10. find - Search for files in a directory hierarchy (-name for specifying filenames)

File Viewing and Editing: ðŸ”—

  1. cat - Concatenate and display file content
  2. less - View file content interactively
  3. head - Display the beginning of a file (-n for number of lines)
  4. tail - Display the end of a file (-n for number of lines, -f for following)
  5. nano - Simple text editor for file editing

Text Manipulation: ðŸ”—

  1. grep - Search text using patterns (-i for case insensitive, -r for recursive)
  2. sed - Stream editor for text manipulation
  3. awk - Text processing tool for pattern scanning and processing
  4. cut - Select sections from each line of files (-d for delimiter, -f for fields)
  5. sort - Sort lines of text files (-n for numerical sort, -r for reverse)

User Management: ðŸ”—

  1. whoami - Display the current user
  2. sudo - Execute a command as the superuser
  3. su - Switch user or become superuser
  4. useradd - Add a new user
  5. passwd - Change user password
  6. usermod - Modify user account
  7. groupadd - Create a new group
  8. chown - Change file owner and group
  9. chgrp - Changes group.
  10. chmod - Changes file permissions
  11. usermod - Modifies a user
  12. userdel - Deletes a user

Process Management: ðŸ”—

  1. ps - Display current processes (aux for detailed information)
  2. pspsps - Summons cats.
  3. top - Display and manage running processes
  4. kill - Terminate processes by ID or name (-9 for forceful termination)
  5. pgrep - Search for processes by name (-l for long listing)

System Information: ðŸ”—

  1. uname - Print system information (-a for all information)
  2. df - Display disk space usage
  3. du - Display directory space usage
  4. free - Display amount of free and used memory in the system
  5. lscpu - Display CPU information

Network: ðŸ”—

  1. ifconfig - Display or configure network interface parameters
  2. ping - Test network connectivity
  3. netstat - Display network connections, routing tables, interface statistics
  4. ssh - Secure shell for remote login
  5. scp - Securely copy files between hosts
  6. curl - Transfer data from or to a server

Package Management (Debian-based systems): ðŸ”—

  1. apt-get - Package handling utility for installing, updating, and removing packages
  2. dpkg - Package manager for Debian-based systems
  3. apt-cache - Query APT cache
  4. aptitude - High-level interface for package management
  5. snap - Command-line interface for Snap packages

Compression and Archiving: ðŸ”—

  1. tar - Archive files (-x for extract, -c for create, -z for gzip compression)
  2. gzip - Compress or decompress files
  3. zip - Package and compress files

System Maintenance: ðŸ”—

  1. shutdown - Shut down or restart the system (-h now for immediate shutdown)
  2. reboot - Reboot the system
  3. date - Display or set the system date and time
  4. crontab - Schedule commands to be executed periodically
  5. uptime - Display how long the system has been running
